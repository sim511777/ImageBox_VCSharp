클립보드에 카피 기능 추가
open ctrl+o 단축키 할당
pseudo 기본 컬러 더 잘보이는 것으로 수정
pseudo 컬러 상용자가 수정 가능하도록 할것
pseudo offset, 간격 사용자가 수정 가능하도록 할것 : MinVal, MaxVal 값 두개만 입력하면 알아서 단계 나누도록 핢
개별 픽셀 표시 폰트와 일반 표시 폰트 두가지 따로 갈것

========
주요기능
1. 아주 큰 이미지라도 메모리가 허용하는 한 로드할 수 있을것
2. 마우스 zoom, pan 기능
3. 확대시 픽셀값 표시 할것
4. 마우스 이동시 커서의 픽셀값과 좌표정보 표시 할것
5. 이미지 픽셀 좌표 단위로 Draw 할 수 있을것
6. 더블버퍼링을 사용해서 깜빡임과 버벅임을 없엘것
7. 일반 컨트롤처럼 동작하게 할것
부가기능
1. 배경색 변경 가능
2. 폰트 변경 가능
3. 중앙선 표시
4. 이미지 보간 기능
5. 이미지 표시할 때 병렬처리
6. 모든 옵션을 항상 바꿀수 있게 매직키 만들것 (ctrl + 더블클릭)
